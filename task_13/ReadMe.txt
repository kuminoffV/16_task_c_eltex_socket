Написать эхо-сервер на сырых сокетах (на стороне клиента) с протоколом UDP,IP,Ethernet
(сырой сокет реализован только на стороне клиента);

Сборка: make

Запуск: 
первый терминал: sudo ./serv
Вторйо терминал: sudo ./client
exit - выход
